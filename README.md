## Deployment

This site is deployed via **Netlify** connected to this GitHub repository.
Every push to `main` branch automatically redeploys the live site.

## To Update the Site

1. Edit the HTML file locally
2. `git add .`
3. `git commit -m "describe your change"`
4. `git push`
5. Netlify auto-deploys within 30 seconds
